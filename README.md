🗺️ A Windows desktop app built with C# and WPF that calculates driving routes within Florida only. Uses Mapbox GL JS for map rendering and OSRM for routing. To use, add your Mapbox public token (pk.) in MainWindow.xaml.cs line 16.


How to add it:

Go to github.com/kevinkactive-droid/FloridaRoute
Click the ⚙️ gear icon next to "About" on the right side
Paste the description above
Click Save changes
